How It Works

This network creates a simple oscillatory pattern based on the two-neuron model described by Beer (1995). The oscillation emerges from the specific pattern of connection weights, which create a feedback loop between the two neurons.

The dynamics follow this continuous-time equation:
τy' = -y + Wσ(g*(y+ θ)) + I

For a detailed explanation of CTRNNs, see Randy Beer's work on continuous-time recurrent neural networks and dynamical approaches to cognitive science.
